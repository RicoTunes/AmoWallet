# Notification Sounds

This directory contains notification sound files for the crypto wallet app.

## Required Sound Files

You need to add the following MP3 sound files to this directory:

1. **notification.mp3** - General notification sound
2. **incoming.mp3** - Incoming transaction sound
3. **success.mp3** - Success/confirmation sound
4. **error.mp3** - Error/failed transaction sound
5. **warning.mp3** - Warning notification sound

## Where to Get Sounds

You can get free notification sounds from:
- https://mixkit.co/free-sound-effects/notification/
- https://freesound.org/
- https://www.zapsplat.com/

## Sound Requirements

- Format: MP3
- Duration: 1-3 seconds
- Size: Keep under 100KB each
- Volume: Normalized to prevent being too loud

## Temporary Solution

The app will work without these files, but you won't hear notification sounds until you add them. The notification system will still work visually.
